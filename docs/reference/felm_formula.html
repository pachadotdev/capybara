Build a felm / feglm formula from parts

Description

Constructs a |-separated formula accepted by felm() and feglm() from a base y ~ x formula plus optional fixed-effects and cluster components supplied as plain character strings. This avoids calling update.formula(), which cannot handle the | syntax and raises "+ not meaningful for factors" errors.

Usage

felm_formula(base, fe = NULL, cluster = NULL)

Arguments

base A one-sided or two-sided formula object or a character string that can be coerced to one, e.g. "ltrade ~ bothin + onein + gsp".
fe Optional character string naming the fixed-effect variable(s), e.g. "year" or "country + year". Omit or pass NULL for models without fixed effects.
cluster Optional character string naming the cluster variable(s), e.g. "pair" or "ctry1 + ctry2". Omit or pass NULL for models without explicit clustering.

Value

A formula of the form y ~ x, y ~ x | fe, or y ~ x | fe | cluster depending on which parts are supplied.

Examples

rhs <- "ltrade ~ bothin + onein + lrgdp"

felm_formula(rhs)
#> ltrade ~ bothin + onein + lrgdp

felm_formula(rhs, fe = "year")
#> ltrade ~ bothin + onein + lrgdp | year

felm_formula(rhs, fe = "year", cluster = "ctry1 + ctry2")
#> ltrade ~ bothin + onein + lrgdp | year | ctry1 + ctry2
